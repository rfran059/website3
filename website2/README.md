# Engineering Portfolio

A high-performance, single-page engineering portfolio website featuring a "Framer-style" aesthetic. Designed specifically for a mechanical engineer specializing in aerodynamics.

## Features

- **Modern Aesthetic**: Framer-inspired design with high-fidelity motion, smooth scrolling, and a minimalist layout.
- **Smooth Scrolling**: Integrated **Lenis** for a premium, inertial scrolling experience.
- **Advanced Animations**: Utilizes **GSAP (GreenSock Animation Platform)** and **ScrollTrigger** for smooth "reveal on scroll" effects and parallax movement.
- **Dynamic Background**: A semi-active, reactive radial gradient background that subtly responds to the user's scroll progress.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile viewing using **Tailwind CSS**.
- **Technical Showcase**: Structured sections for:
    - **About**: Highlights technical software expertise (ANSYS, SolidWorks, etc.), certifications, and academic coursework.
    - **Experience**: A professional timeline of engineering roles, including Formula SAE involvement.
    - **Projects**: A grid-based showcase of engineering achievements and technical implementation details.
    - **Contact**: Direct links for professional networking and inquiries.

## Tech Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Utility-first styling for rapid, responsive development.
- **GSAP & ScrollTrigger**: High-end web animations and scroll-driven interactions.
- **Lenis**: Smooth-scrolling library.
- **Google Fonts**: Modern, legible typography (Inter).

## Setup & Deployment

This website is a static site and can be hosted easily on:
- **GitHub Pages**
- **Vercel**
- **Netlify**

To view locally, simply open `index.html` in any modern web browser.
